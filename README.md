# Odin Rock Paper Scissors

A fully functional implementation of the classic **Rock Paper Scissors** game with a modern user interface. This project was developed as part of [The Odin Project's Foundations Curriculum](https://www.theodinproject.com/) to practice DOM manipulation and event handling in JavaScript.

[Live Demo](https://felixjackquinkwokkenzi.github.io/odin-rock-paper-scissors/)

## 🚀 About The Project

This game features an interactive web UI where the player competes against an automated computer opponent. The match is played until one of the players reaches 5 points.

This project was a major milestone in mastering intermediate JavaScript concepts, transitioning from a console-based application to a fully interactive web application focusing on **DOM manipulation, event listeners, and UI state management**.

## 🎮 How to Play

1. Clone or download this repository to your local machine.
2. Open `index.html` in any web browser.
3. Click on the corresponding buttons (Rock, Paper, or Scissors) to make your choice.
4. The game will announce the result of each round and update the scores dynamically.
5. First player to reach 5 points wins the match! You can click "Play Again" to restart.

## ✨ Key Features & Technical Details

- **Modern UI Design**: Features a beautiful, dark-themed responsive layout.
- **Interactive Gameplay**: Users interact via UI buttons instead of browser prompts, with visual feedback upon hovering and clicking.
- **Dynamic DOM Updates**: Scores and round results are rendered to the DOM dynamically without page reloads.
- **Game State Management**: The game properly manages its state, disabling input once a winner is decided and providing a way to reset the game to its initial state.
- **Randomized Computer Opponent**: A robust random generator (`getComputerChoice`) dynamically returns either "rock", "paper", or "scissors".

## 🛠️ Built With

- **HTML5**: For semantic structure.
- **CSS3**: For styling, flexbox layout, and interactive states.
- **JavaScript (ES6+)**: Core logic, DOM manipulation, and event handling.
